# **App Name**: ITLC India: Dream Homes Reimagined

## Core Features:

- Interactive 3D Villa Model: Display a rotating 3D model of a modern villa that transitions from wireframe to fully rendered on scroll.
- Service Card 3D Tilt Animation: Implement 3D tilt hover animation on service cards to enhance user interaction.
- Construction Progress Animation: Showcase a 3D construction progress animation with smooth, scroll-triggered transitions and parallax effects.
- Project Gallery with 3D Hover Effect: Create a project gallery where images have a 3D hover depth effect. Clicking opens a fullscreen modal with details.
- Lead Generation Form: A lead generation form which helps in collecting the Name, Phone, Plot Size, City, Budget, of users which is submitted and handled with an appropriate back-end. Includes address look-up functionality with LLM-based tool that intelligently suggests and corrects address entries for better data accuracy.

## Style Guidelines:

- Background: Pure white (#FFFFFF) to maintain a clean and luxurious feel.
- Primary: Deep Navy (#191970) for headlines and primary text elements, evoking trust and sophistication.
- Accent: Soft Gold (#C8A96A) for highlights, buttons, and interactive elements, providing a touch of luxury and elegance.
- Font: 'Poppins', a modern sans-serif font, will be used for headlines and body text to maintain a bold and professional aesthetic. (Note: currently only Google Fonts are supported.)
- The layout should be responsive, ensuring optimal viewing experience across desktop and mobile devices. Minimal aesthetics will be employed throughout.
- Employ smooth, premium animations for 3D transitions, hover effects, and scroll-triggered events, enhancing the interactive experience without being flashy.
- Use minimal icons with a subtle 3D depth effect for the service cards and feature boxes.
